# Awesome Identity Theme for Hugo


## Usage

```bash
$ mkdir themes
$ cd themes
$ git clone https://github.com/posquit0/hugo-awesome-identity
```


## Configuration

Add these parameters to your `config.toml`:

```toml
[params]
```
